# Hyprland Dotfiles

My Hyprland setup on CachyOS.

## Components

- Window Manager: Hyprland
- Bar: Waybar
- Launcher: Rofi
- Terminal: Kitty + Starship
- Shell: Fish
- Theme: Kvantum + GTK
- File Manager: Thunar

## Dependencies

Install from Package Installers Beforehand:

- Starship
- Waybar
- Rofi
- Fastfetch
- Puslseaudio
- Thunar
