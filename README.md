# Hyprland Dotfiles

My Hyprland setup on CachyOS.

## Components

- Window Manager: Hyprland
- Bar: Waybar
- Launcher: Rofi
- Terminal: Kitty
- Shell: Fish
- Theme: Kvantum + GTK

![setup](screenshots/setup.png)

## Installation

Clone the repo:

git clone https://github.com/shubhang-10/hyprland-dotfiles.git

Then copy configs:

cp -r config/* ~/.config/
